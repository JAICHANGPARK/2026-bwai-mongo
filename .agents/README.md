# Antigravity Rules and Workflows

이 폴더는 Google Antigravity에서 workspace rules와 workflows로 사용할 참가자용 설정입니다.

## Rules

- `rules/session-1-common-rules.md`: Session 1 전체에 항상 적용할 공통 규칙

## Workflows

- `workflows/session-1-01-gemini-api.md`: Gemini API로 hosted Gemma 4 호출 코드 생성
- `workflows/session-1-02-ollama-server-api.md`: Ollama 로컬 서버 API 호출 코드 생성

Antigravity에서 이 저장소 루트를 열면 workspace 설정으로 사용할 수 있습니다. UI에서 직접 만들 경우에는 위 파일 내용을 각각 Rules와 Workflows에 붙여넣으면 됩니다.
